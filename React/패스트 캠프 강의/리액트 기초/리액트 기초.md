# 리액트 기초

### 라이브러리 vs 프레임워크

라이브러리 - 개발의 편의를 위한 도구 -> 높은 자유도

프레임워크 - 기반까지 갖추진 공장



### 장점

관련 라이브러리가 많고

- swr, framer motion ...

문제해결방법 찾기 쉽고

나와 같은 고민을 할 또는 했던 사람이 많고

실무에서 사용할 확률도 높다.

#### 기술적 장점이있다.

- virtual DOM
- JSX
- Flux
- 함수 컴포넌트

그렇기 대문에 새로운 기술을 빠르게 익히는 능력이 중요하다.
