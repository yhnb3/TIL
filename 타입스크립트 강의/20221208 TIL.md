### 타입스크립트

자바 스크립트는 동적 타입이다. 컴파일 할때는 문제 없지만 런타임에서 문제를 일으킨다.

타입스크립트는 정적 타입이다. 컴파일 단계에서 문제를 일려준다.

### interface VS type alias의 차이

#### 확장하기

interface : extends로 확장

type : &로 확장

#### 기존 타입에 새 필드 추가하기

interface : 그냥 새로 똑같이 한번더 지정하면 새필드가 생긴다.

type : 안된다.

#### 원시타입 선언

interface : 안된다.

type : 된다.
